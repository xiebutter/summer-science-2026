**transformer info**

1) transformer_MLX.py vs transformer_pytorch.py  
- transformer_MLX.py runs on Apple's MLX library for machine learning while transformer_pytorch.py runs on industry-standard PyTorch. Many of the syntax is the same, just instead of using "import torch" use "import mlx.core as mx" and replace the pretext "torch.function" with "mx.function"
- transformer_MLX.py includes a parameter called "dimensions" which is for loading datasets, whether your dataset has one or two dimensions
  - 2 dimensions would be a dataset where each row must be its own block (user/assistant datasets)
  - 1 dimension is just a single long string of text (any .txt file)
  - 2 dimension training does NOT work with tiktoken tokens, finetuning dataset must be tokenized with bpe_tokenizer.py
- for MLX documentation: https://ml-explore.github.io/mlx/build/html/index.html
- transformer_pytorch.py only runs with 2-dimensional datasets, works as expected
- both transformer architectures based on Andrej Karpathy's video: https://www.youtube.com/watch?v=kCc8FmEb1nY

2) token files
- hamlet_token.json --> tokenized Hamlet by Shakespeare
- shakespeare_token.json --> tokenized TinyShakespeare dataset by Andrej Karpathy, also includes special tokens for user/assisstant finetuning
- tinystories_tokens.json --> tokenized validation split of HuggingFace TinyStories dataset (very small portion), includes user/assistant special tokens as well

3) other files
- model_weights.pth are weights generated by transformer_pytorch.py when trained on the Dolly 15k Databricks dataset (user/assistant format)
  - critical error on my part, i pre-trained the model using this dataset --> the model answers in correct format but has no grammar or semantic understanding
- model_weights.safetensors are weights generated by transformer_MLX.py when trained on the first 500,000,000 characters of HuggingFace's TinyStories dataset
- .safetensors files can be loaded into MLX models while .pth files are for PyTorch models
- tinystories_first_quarter.txt --> first 500 million characters of TinyStories
